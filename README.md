# Employee Management App

web application developed in **PHP** for employee management, using **SQLite** as the database.

This project was created as part of my learning process, focusing on clean code, good practices, and continuous improvement based on feedback.

---

## Project Objective

The main goal of this project is to practice and demonstrate:

- PHP fundamentals
- CRUD operations
- Database interaction using SQLite
- Code refactoring and readability
- Basic best practices and standards
- Version control with Git and GitHub

---

## Technologies Used

- PHP
- SQLite
- HTML
- CSS
- PDO (Prepared Statements)

---

## Features

- List employees
- Create new employees
- Edit employee information
- Delete employees
- Assing employees to departments

---

## Recent Improvements

This projects was refactored based on community feedback: 

- Renamed files, variables, and functions to **English**
- Improved variable naming for better readability
- Fixed HTML and PHP bugs
- Applied basic **PSE-12** formatting rules
- Improved SQL usage with prepared statements
- Better consistency across files
